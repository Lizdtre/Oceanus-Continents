# Oceanus + Continents 

Enables using [Oceanus](https://modrinth.com/datapack/oceanus) and [Continents](https://modrinth.com/datapack/continents) simultaneously.

## Load order

Datapack load order should be as follows:

1. Oceanus + Continents
2. Oceanus
3. Continents
